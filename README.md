# 📊 Ames Housing — Advanced Data Analysis

## 📌 Overview
This project presents an advanced multi-dimensional analysis of the Ames Housing dataset using Python, Pandas, and statistical techniques.

The goal is to go beyond basic descriptive statistics and uncover meaningful insights about the factors that influence house prices.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Explore relationships between variables
- Apply advanced analytical techniques
- Generate high-quality, real-world insights

---

## 🗂 Dataset
- **Name:** Ames Housing Dataset  
- **Size:** 2,930 rows × 80+ features  
- **Content:** Residential housing data including price, size, quality, location, and construction details  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔍 Analysis Workflow

### 1. Data Cleaning
- Removed columns with excessive missing values  
- Filled missing numerical values using median  
- Preserved categorical variables for analysis  

---

### 2. Exploratory Data Analysis
- Examined distribution of SalePrice  
- Identified key numerical relationships using correlation  

---

### 3. Advanced Techniques Used

#### ✅ Correlation Analysis
Identified strong relationships between price and features like overall quality and living area.

#### ✅ Cross-Tabulation
Analyzed how categorical variables (e.g., neighborhood, quality) influence house prices.

#### ✅ Percentile Segmentation
Divided houses into price categories to compare characteristics across groups.

#### ✅ Feature Engineering
Created a new metric: **Price per Square Foot** to normalize pricing.

#### ✅ Cohort Analysis
Grouped houses by construction period to analyze time-based trends.

---

## 🧠 Key Insights

- House prices are strongly influenced by **quality, size, and location**
- **Neighborhoods** play a major role in price variation
- Higher-priced homes are both **larger and more expensive per unit area**
- **Newer houses** tend to have higher market value
- Pricing is driven by a combination of multiple interacting factors

---

## 📈 Results
The analysis demonstrates that no single variable determines house price. Instead, pricing is influenced by the interaction of multiple dimensions such as:

- Structural quality  
- Living space  
- Location  
- Modernity
